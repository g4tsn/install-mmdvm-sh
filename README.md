# install-mmdvm-sh
Simple script for install mmdvm software on raspberry PI on Debian 10.

Software installing:
* MMDVMHost
  * Wiring PI for LCD display 16Xn
* DMRGateway
* YSFClients
  * YSFGateway
  * YSFParrot
* ircDDBGateway no gui
* DG9VH MMDVMHost-Dashboard


Read INSTALL.md for exec the script.
